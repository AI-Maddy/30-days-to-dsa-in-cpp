This repository is a complete **30-day DSA journey** in C++ with daily theory, examples, exercises, and solutions.

## 🎯 What makes this memorable

- 🧠 **Memory-first structure**: each day is split into theory + practice.
- 🎨 **Color cues + icons**: visual anchors help faster revision.
- 🧩 **Pattern focus**: reusable problem-solving templates.
- ✅ **Hands-on flow**: examples → exercises → solutions.

## 🗂️ Repository Layout

- `DayXX-*` → daily learning modules
- `cheatsheets/` → fast revision notes
- `all_days_combined.rst` → single consolidated reading file

## 🧪 Source File Standard

Every C++ source file follows a learning template with:

- Function headers
- Pseudocode block
- Explanatory comments
- Minimal, clean `main()` + `solve()` structure

## 📌 Study Loop (recommended)

1. 📖 Read `theory.rst`
2. 👀 Review `examples/`
3. ✍️ Attempt `exercises/`
4. ✅ Compare with `solutions/`

Keep this loop daily to build strong recall and interview confidence.
